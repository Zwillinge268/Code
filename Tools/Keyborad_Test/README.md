# 60%鍵盤簡易測試工具


介紹：工具建立一個60%尺寸的模擬鍵盤。當使用者按下實體鍵盤按鍵時，模擬鍵盤對應的按鍵將變爲藍色。<br>
沒按過的按鍵默認為淺灰色，按過的按鍵為灰色。<br>
